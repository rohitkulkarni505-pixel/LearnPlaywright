# VS Code JavaScript Shortcuts Cheat Sheet

## General

| Shortcut | Action |
|-----------|---------|
| Ctrl + P | Quick Open File |
| Ctrl + Shift + P | Command Palette |
| Ctrl + , | Open Settings |
| Ctrl + B | Toggle Sidebar |
| Ctrl + J | Toggle Terminal |
| Ctrl + Shift + E | Explorer |
| Ctrl + Shift + F | Global Search |
| Ctrl + Tab | Switch Between Files |

---

## Editing

| Shortcut | Action |
|-----------|---------|
| Ctrl + C | Copy Line (if nothing selected) |
| Ctrl + X | Cut Line (if nothing selected) |
| Alt + Up/Down | Move Line Up/Down |
| Shift + Alt + Up/Down | Duplicate Line |
| Ctrl + Enter | Insert Line Below |
| Ctrl + Shift + Enter | Insert Line Above |
| Ctrl + / | Toggle Single-Line Comment |
| Shift + Alt + A | Toggle Multi-Line Comment |
| Ctrl + Space | Trigger IntelliSense |
| Ctrl + Shift + K | Delete Line |
| Ctrl + D | Select Next Occurrence |
| Ctrl + L | Select Current Line |

---

## Multi-Cursor

| Shortcut | Action |
|-----------|---------|
| Alt + Click | Add Cursor |
| Ctrl + Alt + Up/Down | Add Cursor Above/Below |
| Ctrl + D | Select Next Match |
| Ctrl + Shift + L | Select All Matches |

## JavaScript Snippets

### Function

Type:

```js
fn
```

Press **Tab**

```js
function name(params) {

}
```

---

### Arrow Function

Type:

```js
const
```

Then:

```js
const myFunction = () => {

};
```

---

### Console Log

Type:

```js
log
```

Press **Tab**

```js
console.log();
```

---

### For Loop

Type:

```js
for
```

Press **Tab**

```js
for (let i = 0; i < array.length; i++) {

}
```

---

### For Of

Type:

```js
forof
```

Press **Tab**

```js
for (const item of items) {

}
```

---

### If Statement

Type:

```js
if
```

Press **Tab**

```js
if (condition) {

}
```

---

### Try Catch

Type:

```js
try
```

Press **Tab**

```js
try {

} catch (error) {

}
```

## Most Important JS Developer Shortcuts

- Ctrl + Space → IntelliSense
- Ctrl + . → Quick Fix
- F2 → Rename Variable
- F12 → Go to Definition
- Shift + Alt + F → Format Code
- Ctrl + / → Single-Line Comment
- Shift + Alt + A → Multi-Line Comment
- Ctrl + D → Multi-Select
- Alt + Up/Down → Move Line
- Shift + Alt + Up/Down → Duplicate Line
